# DemoAdvangeGonlineshopping
Automatización web
El objetivo de este reto es validar sus conocimientos en automatización web y su capacidad de
autoaprendizaje. Debe crear un proyecto que cumpla con los siguientes criterios:
• Uso del patrón Page Object Model (P.O.M) o ScreenpIay.
• Lenguaje de programación Java
• Proyecto construido con Gradle
• Uso de JUnit
• Automatización con Selenium
• Uso de Serenity WEB
