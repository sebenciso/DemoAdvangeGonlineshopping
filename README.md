# DemoAdvangeGonlineshopping
Automatización web
El objetivo de este reto es validar sus conocimientos en automatización web y su capacidad de
autoaprendizaje. Debe crear un proyecto que cumpla con los siguientes criterios:
• Uso del patrón Page Object Model (P.O.M) o ScreenpIay.
• Lenguaje de programación Java
• Proyecto construido con Gradle
• Uso de JUnit
• Automatización con Selenium
• Uso de Serenity WEB

 Instalar y configurar en el equipo:

- Java JDK 8 o superior.
Paso 1: Descargar el instalador de Java JDK

Lo primero que debes hacer es descargar el instalador de Java JDK 8 o superior desde el sitio oficial de Oracle. Puedes encontrar el enlace de descarga aquí: https://www.oracle.com/java/technologies/javase-downloads.html

Asegúrate de descargar la versión adecuada para tu sistema operativo y arquitectura (32 bits o 64 bits).

Paso 2: Ejecutar el instalador de Java JDK

Una vez que hayas descargado el instalador, debes ejecutarlo y seguir las instrucciones que aparecen en pantalla para instalar Java JDK en tu equipo.

Es probable que se te solicite aceptar los términos y condiciones de Oracle y que elijas la ubicación donde deseas instalar Java en tu equipo.

Paso 3: Configurar las variables de entorno de Java

Una vez que hayas instalado Java JDK en tu equipo, debes configurar las variables de entorno de Java para poder utilizarlo en la línea de comandos.

Para hacerlo, sigue los siguientes pasos:

Abre el panel de control de Windows.
Busca la opción "Sistema" y haz clic en ella.
Haz clic en "Configuración avanzada del sistema" en la barra lateral izquierda.
Se abrirá una ventana de propiedades de sistema. Haz clic en la pestaña "Opciones avanzadas".
Haz clic en el botón "Variables de entorno".
En la sección "Variables del sistema", busca la variable "Path" y haz clic en el botón "Editar".
Agrega el directorio de instalación de Java JDK (por ejemplo, "C:\Archivos de programa\Java\jdk1.8.0_281\bin") al final del valor existente de la variable "Path", separado por un punto y coma (;).
Haz clic en "Aceptar" para guardar los cambios.

Paso 4: Verificar la instalación de Java JDK.

Para verificar que Java JDK se ha instalado correctamente en tu equipo, abre una ventana de línea de comandos y escribe el siguiente comando:

Oprimir la tecla Windows +R:"cmd" y digitar: java -version

Este comando debe mostrar la versión de Java JDK que has instalado. Si el comando no funciona, revisa los pasos anteriores para asegurarte de haber configurado correctamente las variables de entorno de Java.

Paso 6: Agregar las dependencias necesarias en el archivo build.gradle:

- Agregar Gradle
- Selenium WebDriver
